# hugo.ianteda.com

Hugo static website for www.ianteda.com

```bash
hugo server --noHTTPCache --disableFastRender  
```

Demo theme

cd in theme example dir

```bash
cd theme/awesome/exampleSite
```

Run hugo server with theme

```bash
hugo serve  --themesDir ../..
```
